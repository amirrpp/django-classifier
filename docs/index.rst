django-classifier
=============================================

Flexible constructor to create dynamic list of heterogeneous properties for
some kind of entity. This set of helpers useful to create properties like
contacts or attributes for describe car/computer/etc.

You must create your own models for each kind of entity but it gives more
flexibility, more simpler and allow to easily change default behavior.

read :doc:`tutorial` to see all picture of idea.
